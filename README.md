## Local Setup

### 1. Clone

```bash
git clone https://github.com/karnapuumamaheswari/portfolio.git
cd portfolio
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run development server

```bash
npm run dev
```

### 4. Build for production

```bash
npm run build
```

## Deployment

This project is configured for GitHub Pages.

```bash
npm run deploy
```
